Food Recipe App
The Food Recipe App is a mobile application designed to help users discover, save, and share recipes. It provides a user-friendly interface for browsing recipes, viewing details such as ingredients and cooking instructions, and saving favorite recipes for later use.

Features:
Browse a wide variety of recipes
View detailed information for each recipe, including ingredients and cooking instructions
Save favorite recipes for quick access
Share recipes with friends and family
Search for recipes by name, category, or ingredients
Rate and review recipes

Technologies Used:
Swift: The primary programming language used for iOS app development.
UIKit: The framework used for building the app's user interface.
Realm: Used for local data storage and management.
SDWebImage: Library for asynchronous image loading and caching.
DatabaseManager: Custom database manager for handling recipe data.
Firebase firestore: To save realtime user and recipe data.
MVVM Architecture: Design pattern used to organize code and separate concerns.

Installation:
To run the Food Recipe App locally, follow these steps:
  Clone this repository to your local machine.
  Open the project in Xcode.
  Build and run the app on a simulator or a physical device.

Usage:
Upon launching the app, you'll be presented with a list of recipes.
Tap on a recipe to view its details, including ingredients and cooking instructions.
Use the save button to add a recipe to your favorites.
Share recipes with others using the share button.
Use the search feature to find recipes based on name, category, or ingredients.
